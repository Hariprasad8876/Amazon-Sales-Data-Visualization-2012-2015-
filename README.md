📦 Amazon Data Visualization (2012–2015)
📌 Project Overview

This project focuses on analyzing Amazon sales data from 2012 to 2015 using Power BI. The goal is to uncover sales trends, customer behavior, and business insights through interactive dashboards and visual reports.

🎯 Objective

Analyze historical sales data to identify trends

Understand customer purchasing patterns

Track revenue growth and product performance

Support business decisions with data-driven insights

⚙️ Tools & Technologies

Power BI (Dashboard & Visualization)

Power Query (Data Cleaning & Transformation)

DAX (Calculated Measures & KPIs)

Excel / CSV (Dataset)

📊 Key Features

Year-wise and month-wise sales analysis

Region-wise performance visualization

Top-selling products and categories

Profit and revenue trend analysis

Interactive filters and drill-down reports

📈 Insights & Outcomes

Identified high-performing regions and product categories

Observed seasonal sales trends

Highlighted profit variations across different segments

Enabled better strategic planning through visual insights

🚀 Future Enhancements

Integration with real-time sales data

Advanced forecasting using machine learning

Customer segmentation analysis

Automated dashboard updates
